# Sentiment Analysis Tool

This is a **Streamlit-based Sentiment Analysis web app** that allows users to analyze text and get sentiment predictions (positive, negative, neutral).  

The app uses **Hugging Face Transformers** under the hood to perform sentiment classification.

---

## 🔗 Live Demo

You can try the app online here:  
[Sentiment Analysis Tool on Hugging Face Spaces](https://huggingface.co/spaces/Tumisang555/Sentiment-Analysis-tool)

---

## ⚡ Features

- Input text and get sentiment predictions in real-time  
- Lightweight and fast thanks to Streamlit and pre-trained models  
- Automatically downloads required model from Hugging Face Hub
